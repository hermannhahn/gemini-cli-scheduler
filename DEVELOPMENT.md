# Gemini CLI Scheduler Extension

Development instructions for this project.

## Scripts

- `npm run lint`: Run linter to check for code style and quality issues.
- `npm run test`: Run tests to ensure code functionality.
- `npm run webpack`: Build the extension and check for build errors.
- `npm run update:version`: Update the version in all necessary files, run tests and lint before updating, and commit and push changes.
- `npm run deploy`: Deploy the extension by changing to `main` branch, pulling the latest changes, running lint, creating and pushing a tag, creating and pushing a release on GitHub, updating the local extension, and returning to `development` branch.
